# PLOX
A speedy interpreter written in C++ for the Lox programming language. Based on the language from the book Crafting interpreters
by Bob Nystrom!

How To Build
---

Run CMake in build dir
```sh
# I recommend making build dir in the project home dir
cmake <src-dir>

# then build
make plox
```
